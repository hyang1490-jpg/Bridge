# Bridge — 中日AI生态翻译器

> 哪个AI最适合你的业务？6模型×4场景，基于真实Benchmark测试数据的交互式模型选型工具。

## What It Does

日本企业想用AI但不知道选哪个模型。Bridge让用户选择业务场景和约束条件，输出最适合的模型推荐方案。

## Data Source

基于作者亲自执行的6模型（Gemini Pro, Claude Opus, Qwen 2.5, Grok Heavy, Gemini Flash, DeepSeek V3）×4场景（客服邮件、新闻摘要、数据处理、商务翻译）Benchmark测试，控制变量为同一Prompt无修改投喂全部模型。

## Key Insights

1. 没有全场景最强模型，企业应按场景混合部署
2. DeepSeek达到80%质量但成本仅为Claude的1/10
3. Qwen的日语IT用语精准度超过Gemini和Grok

## Tech

React + Tailwind, 纯前端, 无后端依赖
